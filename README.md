# Auto Close New Tab Page

A minimal Chrome extension that closes a new tab page as soon as it loses focus.

Behavior:

- Opening a new tab page and switching to any other tab closes that new tab page.
- Opening another new tab page closes any other open new tab pages, so only one new tab page remains.
- If you type or navigate away from the new tab page before switching tabs, the extension stops treating that tab as a new tab page.
